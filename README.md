# Ecommerce Full Stack Project

## Overview
This is a full stack ecommerce web application developed using modern web technologies.  
It allows users to browse products, add items to cart, and manage orders.

---

## Tech Stack
### Frontend:
- React / Angular
- HTML, CSS, JavaScript

### Backend:
- .NET Web API

### Database:
- SQL Server

---

##  Features
-  User Authentication (Login/Register)
-  Product Listing
-  Add to Cart
-  Order Management
-  Search & Filter (if implemented)

---

## Project Structure
Ecommerce-Project/
 ├── frontend/   # UI (React/Angular)
 └── backend/    # API (.NET)

---

##  How to Run the Project

### Frontend Setup
cd frontend  
npm install  
npm start  

---

### Backend Setup
cd backend  
dotnet restore  
dotnet run
