# Ecommerce Full Stack Project

## Project Overview
This is a full stack ecommerce web application with frontend and backend integration.

## Tech Stack
- Frontend: React / Angular
- Backend: .NET Web API
- Database: SQL Server

## Features
- User authentication (Login/Register)
- Product listing
- Add to cart
- Order management

## Project Structure
Ecommerce-Project/
 ├── frontend/
 └── backend/

##  How to Run

### 1️ Frontend
cd frontend
npm install
npm start

### 2️ Backend
cd backend
dotnet restore
dotnet run
